# Apple_forcasting_project-
Predicting the next 30 days of Apple Inc. (AAPL) stock prices using Time-Series Forecasting. This project was developed during my Data Science internship at Aivariant
Tech Stack
Language: Python
Model: SARIMA (Seasonal AutoRegressive Integrated Moving Average)
Deployment: Streamlit Dashboard
Data Sources: Yahoo Finance (Stock Prices), FRED (Macroeconomic indicators)
How to Run
Install requirements: pip install -r requirements.txt
Launch the app: streamlit run app.py
